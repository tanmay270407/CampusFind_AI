# **App Name**: CampusFind AI

## Core Features:

- Lost Item Upload: Allows students to upload photos and descriptions of lost items, including item type, color, brand, and location lost.
- Found Item Upload: Enables staff/security to upload photos and location details of found items.
- AI Image Matching: Uses AI to find visually similar items between lost and found uploads and ranks the top matches.
- Location Tracking: Records the location where each item was found to aid in the matching process and assist owners in locating their items.
- Match Notifications: Sends alerts to users when a potential match is found for their lost or found items.
- Claim System: Implements a claim system requiring proof to ensure rightful owners retrieve their items; the AI tool provides additional relevant details to inform the reviewer.
- User Authentication: Ensures only campus users can access the platform, providing a secure login system.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke trust and calm.
- Background color: Light blue (#E3F2FD) for a clean, unobtrusive feel.
- Accent color: Yellow-orange (#FFAB40) to highlight important CTAs.
- Body and headline font: 'PT Sans' for a modern, readable feel.
- Use clear, minimalist icons to represent item types and actions.
- Clean, card-based layout for item displays, ensuring a responsive design.
- Subtle animations on match notifications and when confirming claims to enhance user experience.
